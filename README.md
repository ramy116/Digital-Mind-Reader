# Digital Mind Reader: Enigma
A Python-based interactive "Mind Reading" game built with **Pygame**.

## 🔮 The Trick
The program uses the classic 21-card mathematical trick. By sorting items into three groups and asking the user to identify their group three times, the algorithm uses binary logic to isolate the specific item the user is thinking of.

## 🚀 Features
- **Dynamic Themes**: Choose between Numbers, Fruits, or Names.
- **Modern UI**: Dark-mode aesthetic with "Glassmorphism" UI elements.
- **Animations**: Real-time particle system and pulsating background effects.
